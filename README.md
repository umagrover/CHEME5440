## Recreation of Mazor et al. Model
This code will be our recreation of the original scenario in the Mazor et al. paper using parameters from Quantitative Fundamentals of Molecular and Cellular Bioengineering. It will use the ODEs used in Mazor et al.

## Updated model with Wormlike Chain Linker
This code will show our modified scenario using the wormlike-chain model to show model differences with a flexible polypoptide linker. It will use updated ODEs to reflect the use of the wormlike-chain model.
